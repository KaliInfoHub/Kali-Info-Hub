What is Kali Linux?

Kali Linux is a security-focused Linux distribution developed by Offensive Security. It comes preloaded with hundreds of powerful penetration testing and digital forensics tools.

System Requirements

  • 2+ GHz dual-core processor
  • 2 GB RAM minimum (4 GB recommended)
  • 20 GB of disk space
  • USB drive (for Live boot) or virtual machine software

Download Kali

Get the latest ISO image from:

https://www.kali.org/get-kali/

Install Options

1. Live USB Boot (Recommended for Beginners)

  • Use balenaEtcher or Rufus to write the ISO to a USB stick
  • Boot from USB and run Kali in live or install mode

2. Virtual Machine (VMware/VirtualBox)

  • Download the Kali VM image (pre-built VMs are available)
  • Import into VirtualBox or VMware
  • Enable networking and guest additions

3. Kali on Windows (WSL)

wsl --install -d kali-linux

Runs Kali from within Windows using the Windows Subsystem for Linux.

Post-Installation Tips

  • Update your system:
    • sudo apt update && sudo apt upgrade
  • Enable root login (optional):
    • sudo passwd root
  • Use kali-undercover to make Kali look like Windows for stealth mode
    • kali-undercover

Start Exploring

Once installed, open the terminal and start testing tools like nmap, hydra, or launch burpsuite to begin exploring ethical hacking.
